#### 二、调整网卡中断合并(interrupt coalescing)参数
```bash
ethtool -C eth20 rx-usecs 64 rx-frames 128
ethtool -C eth21 rx-usecs 64 rx-frames 128
```

#### 三、中断率的计算

1） **已知条件**

  通过sar命令观察对应的PPS:

  ```text
  每 NIC PPS: ~6M（总 12M / 2 张卡）
  每 NIC 队列数: 63
  每队列 PPS: 6,000,000 / 63 ≈ 95,000 pps
  每队列包间隔: 1,000,000 µs / 95,000 ≈ 10.5 µs
  ```

2）**rx-usecs=3 时**

  ```text
  第一个包到达 → 启动 3 µs 计时器
    ↓ 3 µs 后
  窗口内只有 ~0.3 个额外包 → 实际只处理 1 个包
    ↓
  中断频率 ≈ PPS ≈ 95,000 irq/sec/队列
  总中断 ≈ 126 队列 × 95,000 ≈ 12M irq/sec
  ```

3) **rx-usecs=64 时**

  ```text
  第一个包到达 → 启动 64 µs 计时器
    ↓ 64 µs 后
  窗口内到达 ~6 个包（64 / 10.5 ≈ 6）
    ↓
  中断频率 = 95,000 / 6 ≈ 15,800 irq/sec/队列
  总中断 ≈ 126 × 15,800 ≈ 2M irq/sec
  ```

4) **实际降幅**
  ```text
  12M → 2M = 约 6 倍
  ```
