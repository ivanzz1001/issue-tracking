# 网卡中断合并


#### 查看网卡中断合并参数

```bash
# ethtool -c eth20
Coalesce parameters for eth20:
Adaptive RX: off  TX: on
stats-block-usecs: n/a
sample-interval: n/a
pkt-rate-low: n/a
pkt-rate-high: n/a

rx-usecs: 64
rx-frames: 128
rx-usecs-irq: n/a
rx-frames-irq: n/a

tx-usecs: 8
tx-frames: 128
tx-usecs-irq: n/a
tx-frames-irq: n/a

rx-usecs-low: n/a
rx-frame-low: n/a
tx-usecs-low: n/a
tx-frame-low: n/a

rx-usecs-high: n/a
rx-frame-high: n/a
tx-usecs-high: n/a
tx-frame-high: n/a

CQE mode RX: on  TX: off
```

#### 调整网卡中断合并(interrupt coalescing)参数
```bash
ethtool -C eth20 rx-usecs 64 rx-frames 128
ethtool -C eth21 rx-usecs 64 rx-frames 128
```
