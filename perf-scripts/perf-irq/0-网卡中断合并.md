#### 一. 查看网卡中断合并参数

```bash
# ethtool -c eth20
Coalesce parameters for eth20:
Adaptive RX: off  TX: on
stats-block-usecs: n/a
sample-interval: n/a
pkt-rate-low: n/a
pkt-rate-high: n/a

rx-usecs: 64
rx-frames: 128

rx-frames-irq: n/a

tx-usecs: 8
tx-frames: 128

tx-frames-irq: n/a

rx-usecs-low: n/a
rx-frame-low: n/a
tx-usecs-low: n/a
tx-frame-low: n/a

rx-usecs-high: n/a
rx-frame-high: n/a
tx-usecs-high: n/a
tx-frame-high: n/a

```

#### 2. 调整网卡中断合并(interrupt coalescing)参数
```bash
ethtool -C eth20 rx-usecs 64 rx-frames 128
ethtool -C eth21 rx-usecs 64 rx-frames 128
```

#### 3. 中断率的计算

1) **通过sar命令观察对应的PPS**

```text
每 NIC PPS: ~6M（总 12M / 2 张卡）
每 NIC 队列数: 63
每队列 PPS: 6,000,000 / 63 ≈ 95,000 pps
每队列包间隔: 1,000,000 µs / 95,000 ≈ 10.5 µs
```
